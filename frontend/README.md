# Smart Traffic Management Dashboard

React-based dashboard for real-time traffic monitoring and management.

## Features

- Real-time traffic map with heatmaps
- Traffic analytics and charts
- Signal control interface
- Emergency vehicle routing
- Anomaly detection alerts

## Development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Environment Variables

Create `.env.local`:

```
REACT_APP_API_URL=http://localhost:8000
REACT_APP_MAPBOX_TOKEN=your_mapbox_token
```
